## 课程项目要求

- 命令行工具
- 参考git实现原理，实现blob，tree，commit核心存储结构

功能点：

- 可以提交commit，可以进行git log查看commit历史
- 可以进行git reset回滚到指定commit
- 可创建多分支，可在分支之间切换
- 无需实现的功能：merge，远程仓库
